# Contributing to VÉNARO

Thank you for your interest in the VÉNARO Premium E-Commerce Platform! While this is a personal portfolio and professional demonstration project, I value feedback and high-quality contributions.

---

## 🚀 How to Contribute

### 1. Reporting Bugs
- Use the [Issue Tracker](https://github.com/vishal-dev1128/VENARO-Ecommerce-Website/issues).
- Provide a clear, descriptive title and a detailed description with steps to reproduce.
- Attach screenshots if applicable.

### 2. Suggesting Enhancements
- Open an [Issue](https://github.com/vishal-dev1128/VENARO-Ecommerce-Website/issues) with the tag `enhancement`.
- Explain why the feature would be beneficial and how it fits the "Luxury/Premium" theme of the project.

### 3. Pull Requests
- Fork the repository.
- Create a feature branch: `git checkout -b feature/amazing-feature`.
- Ensure your code follows modern PHP 8 standards and PSR-1/PSR-12.
- Commit your changes: `git commit -m 'feat: Add some amazing feature'`.
- Push to the branch: `git push origin feature/amazing-feature`.
- Open a Pull Request.

---

## 🎨 Coding Standards
- Use **CamelCase** for classes and **snake_case** for functions/variables.
- Always use **PDO Prepared Statements** for database interaction.
- Maintain consistent indentation (4 spaces).
- Comment complex logic extensively.

## 🛡️ Security
If you find a security vulnerability, please refer to our [Security Policy](.github/SECURITY.md).

---

Thank you for helping make VÉNARO even better!
